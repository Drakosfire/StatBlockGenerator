# StatBlockGenerator
More organized and intentional build 
This will require greater than 12 GB of VRAM and about 22+ GB of system RAM in my experience. 

This dockerfile is designed to be built inside of a clone of this repo https://github.com/G-Ambatte/homebrewery/tree/experimentalCommandLineBrewProcess 
It's messy and I'll figure out a better way to do it eventually.
Then you will need /models/Speechless-Llama2-Hermes-Orca-Platypus-WizardLM-13B-GPTQ from https://huggingface.co/TheBloke/Speechless-Llama2-Hermes-Orca-Platypus-WizardLM-13B-GPTQ
Then /models/stable-diffusion/SDXLFaetastic_v20.safetensors from https://civitai.com/models/129681/sdxl-faetastic
Finally you will need /model/statblock-alpha you can message me for that since I have not uploaded it publicly yet
